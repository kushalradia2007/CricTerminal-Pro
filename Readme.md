Meraz 6.0 | Gears of Glory
Meraz 6.0 is a high-performance, steampunk-inspired web platform designed for the annual techno-cultural fest of IIT Bhilai. The website provides a seamless experience for participants to explore events, learn about partners, and manage their registrations through a futuristic interface.

🚀 Live Features
Immersive Visuals: A sleek "Dark Mode" aesthetic featuring neon-cyan accents, CSS-grid background animations, and glassmorphism.

AI Assistant: A built-in "Meraz AI" chatbot mascot to assist users with navigation and common queries.

Responsive Design: Fully adaptive layouts for mobile, tablet, and desktop viewing.

🛠 Tech Stack
Frontend: HTML5, CSS3 (Flexbox/Grid), JavaScript (ES6+).

Visuals: Unsplash API for imagery, Mixkit for background video loops, and Giphy for dynamic tech previews.

📂 Project Structure
1. Home Page (hompage.html)
The landing experience featuring:

Hero Video: High-energy background video with a glassmorphism overlay.

Sponsor Showcase: Tiered display (Title, Powered By, Associate) for festival partners.

Meraz AI Chatbot: A functional mascot that responds to queries about events and registration.

2. Events Portal (events.html)
A categorized dashboard for all festival activities:

Categories: Culturals, Sci-Tech, E-Cell, and FinTech.

Interactive Cards: VIDEO PREVIEWS on hover (on some of the events ) and real-time "Register" buttons that.

Sticky Navigation: Quick-access category bar for easy browsing.

3. Registration Portal (registration.html)
A dedicated entry point for new participants:

Futuristic UI: Features an animated "scanning" line and a perspective-distorted grid background.

Secure Intake: Form fields for participant identification and institute email.
